# PromptFlow

一个专为网页AI使用而设计的VS Code扩展，支持在单个Markdown文件中进行多轮提示词编写。

## 🎯 核心功能

- **单文件多轮对话**：在一个Markdown文件中管理多轮AI对话
- **Markdown支持**：使用熟悉的Markdown语法编写提示词
- **一键复制**：快速复制提示词到剪贴板
- **本地管理**：所有提示词本地保存，保护隐私

## 📋 要求

- Visual Studio Code 1.85.0 或更高版本

## 🛠️ 安装

1. 打开 VS Code 扩展面板 (`Ctrl+Shift+X`)
2. 搜索 `PromptFlow`
3. 点击安装

## 🚀 使用

1. 创建新的 `.md` 文件
2. 使用命令面板 (`Ctrl+Shift+P`) 输入 `PromptFlow`
3. 开始编写你的提示词
4. 使用快捷键快速复制内容到网页AI
